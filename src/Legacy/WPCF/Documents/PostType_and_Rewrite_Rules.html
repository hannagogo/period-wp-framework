Adding rewrite rules

1. Create new post type

new PostType(array(
 'name'=> 'post-type-name',
// some args
// :
// :
 'custom_rules' => array(
  'pattern' => 'mapping'
//'(?:/(\d{4}/?)?(?:/(\d{1,2})/?)?(?:/(\d{1,2})/?)?)?/?$' => 'index.php?' . implode('&', array('scy=$matches[1]&scm=$matches[2]&scd=$matches[3]') ),
) ) )

2. Enable query_vars

 enable_query_vars(array("scy", "scm", "scd"));

3. Get value(s)

$y=get_query_var('scy')
